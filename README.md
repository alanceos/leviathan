## Getting Started

1. Clone the repository
```bash
git clone [your-repo-url]
cd leviathan-token
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
